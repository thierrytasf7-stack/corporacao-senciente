## Quick Commands

- `@data-lead` - Coordenador de coleta e processamento
- `@data-scraper` - Coletor de dados esportivos
- `@data-processor` - Processador de dados em tempo real
- `@data-validator` - Validador de qualidade de dados

## Workflow

1. @data-lead coordena coleta de dados
2. @data-scraper coleta odds, stats, markets
3. @data-processor processa e agrega dados
4. @data-validator valida qualidade
5. @data-lead review e decisões
