## Quick Commands

- `@ceo-bet` - Strategic Orchestrator for BET-SPORTS
- `*daily-briefing` - Briefing consolidado de todos os squads
- `*orchestrate` - Coordena squads para execução de objetivo
- `*strategic-decision` - Decisão estratégica de alto nível
- `*portfolio-review` - Review completo do portfólio

## Workflow

1. @ceo-bet recebe objetivo/diretiva
2. Analisa qual(is) squad(s) deve acionar
3. Delega tasks para squads especializados:
   - live-betting: execução de apostas
   - data-sports: pipeline de dados
   - strategy-sports: desenvolvimento de estratégias
   - infra-sports: integrações bookmakers
   - analytics-sports: análise de performance
4. Consolida resultados e decide próximo passo
