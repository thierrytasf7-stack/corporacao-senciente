# Orchestrate Data Sports Squad

## Purpose
Coordena o squad data-sports para pipeline de dados esportivos, garantindo qualidade e disponibilidade de dados para tomada de decisão.

## Key Responsibilities
- Monitorar pipeline de dados esportivos
- Garantir qualidade e integridade dos dados
- Coordenar coleta e processamento de dados
- Otimizar fluxos de dados para análise
- Identificar e resolver problemas de dados

## Workflow
1. **Data Pipeline Review**: Analisa status do pipeline de dados
2. **Quality Check**: Avalia qualidade e integridade dos dados
3. **Collection Coordination**: Coordena fontes de dados e coleta
4. **Processing Optimization**: Otimiza processamento de dados
5. **Analysis Support**: Garante dados disponíveis para análise
6. **Problem Resolution**: Identifica e resolve problemas de dados

## Success Metrics
- Disponibilidade e qualidade dos dados
- Tempo de processamento de dados
- Cobertura e completude dos dados
- Precisão e confiabilidade dos dados
- Satisfação dos squads dependentes

## Dependencies
- data-sports squad
- live-betting squad (para dados em tempo real)
- analytics-sports squad (para análise de dados)
- strategy-sports squad (para dados estratégicos)
