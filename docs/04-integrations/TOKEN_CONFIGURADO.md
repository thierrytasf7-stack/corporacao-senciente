**IMPORTANTE:** O token está armazenado no remote URL do Git. Isso é seguro localmente, mas:

1. **Nunca commite o arquivo `.git/config`** com o token
2. Se precisar compartilhar o repositório, remova o token primeiro:
   ```bash
   git remote set-url origin https://github.com/thierrytasf7-stack/Diana-Corporacao-Senciente.git
   ```
3. O token pode ser revogado a qualquer momento em:
   https://github.com/settings/tokens
