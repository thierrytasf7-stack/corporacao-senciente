# Daily Operations Checklist

Checklist diario para operacoes do sistema de trading.

## Morning (09:00)
- [ ] Health check de todos os sistemas
- [ ] Verificar saldo e posicoes
- [ ] Revisar trades executados overnight
- [ ] Verificar triggers ativos
- [ ] Market scan (tendencia, volume, sentimento)
- [ ] Briefing do CEO

## During Trading Hours
- [ ] Monitorar posicoes abertas a cada 30min
- [ ] Verificar alertas de risco
- [ ] Ajustar stops conforme necessario
- [ ] Monitorar rate limits da API
- [ ] Registrar trades executados

## Evening (18:00)
- [ ] Revisar PnL do dia
- [ ] Atualizar journal de trades
- [ ] Verificar drawdown vs limites
- [ ] Desativar estrategias de scalping (se aplicavel)
- [ ] Portfolio review com CEO
- [ ] Backup de dados/configs

## Weekly (Sexta)
- [ ] Performance review semanal
- [ ] Revisar estrategias ativas (manter/ajustar/desativar)
- [ ] Rebalanceamento se necessario
- [ ] Atualizar limites de risco
- [ ] Revisar logs de erro da semana
