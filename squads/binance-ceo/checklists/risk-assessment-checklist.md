# Risk Assessment Checklist

Checklist de avaliacao de risco para novas operacoes ou mudancas no portfolio.

## Position Risk
- [ ] Risk per trade <= 1% do capital
- [ ] Position size <= 10% do portfolio
- [ ] Stop loss obrigatorio definido
- [ ] Risk/Reward >= 1.5:1
- [ ] Liquidez suficiente para o tamanho da posicao

## Portfolio Risk
- [ ] Concentracao max por ativo <= 20%
- [ ] Numero de posicoes abertas <= limite
- [ ] Correlacao entre posicoes < 0.7
- [ ] Alavancagem total <= 5x
- [ ] Drawdown diario < 5%
- [ ] Drawdown semanal < 10%

## Market Risk
- [ ] Volatilidade atual analisada (ATR)
- [ ] Eventos macro considerados
- [ ] Liquidez do mercado verificada
- [ ] Horario de trading adequado

## Operational Risk
- [ ] API Binance conectada e funcional
- [ ] Trigger monitor operacional
- [ ] Saldo suficiente
- [ ] Margem disponivel (futures)

## Approval Required
- [ ] Risk Manager: APROVADO
- [ ] CEO: INFORMADO (posicao > 5% portfolio)
