# Tech Stack - BINANCE-CEO Squad

## Backend
- **Runtime:** Node.js 18+ / TypeScript 5
- **Framework:** Express.js 4.18
- **Database:** PostgreSQL (via Prisma ORM)
- **Cache:** Redis 4.6 / ioredis 5.3
- **Exchange API:** node-binance-api 1.0.12
- **Auth:** JWT (jsonwebtoken + bcryptjs)
- **Validation:** Joi + AJV
- **Logging:** Winston
- **Jobs:** node-cron 3
- **WebSocket:** ws 8.14
- **Testing:** Jest 29

## Frontend
- **Framework:** React 18 + TypeScript
- **Bundler:** Vite 5
- **State:** Redux Toolkit 2
- **Styling:** Tailwind CSS 3.3
- **HTTP:** Axios 1.6
- **Real-time:** Socket.io-client 4.7
- **Routing:** React-Router-DOM 6
- **Testing:** Vitest + Testing-Library

## Infrastructure
- **Process Manager:** PM2
- **OS:** Windows 11 (native)
- **Scripting:** PowerShell + Python
- **Port Range:** 21340-21341

## Exchange
- **Primary:** Binance (REST + WebSocket)
- **Testnet:** Binance Testnet (sandbox)
- **Markets:** Spot + Futures (USDT-M)
