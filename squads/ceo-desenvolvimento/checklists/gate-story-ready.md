# Quality Gate: Story Ready for Development

## Mandatory (all must pass)
- [ ] Story has clear title and description
- [ ] Acceptance criteria are specific and testable
- [ ] Complexity estimated (fibonacci)
- [ ] Dependencies resolved or documented
- [ ] Branch created (if needed)
- [ ] DB changes identified (if any)

## DB Stories (additional)
- [ ] @data-engineer snapshot taken
- [ ] Schema/migration applied
- [ ] Security audit passed

## Gate Decision
- **READY**: All checks pass → proceed to implementation
- **NOT READY**: Missing criteria → return to Athena/PO
