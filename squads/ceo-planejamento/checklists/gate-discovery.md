# Quality Gate: Discovery Phase

## Mandatory (all must pass)
- [ ] Problem statement clearly defined
- [ ] Target user/persona identified
- [ ] Domain context documented
- [ ] Key assumptions listed

## Standard Mode (3+ must pass)
- [ ] Market landscape analyzed
- [ ] Key competitors identified
- [ ] User pain points documented
- [ ] Feasibility assessment complete
- [ ] Project brief document generated

## Comprehensive Mode (all must pass)
- [ ] Full market research document
- [ ] Competitive analysis with differentiators
- [ ] Brainstorming session conducted
- [ ] Multiple solution directions explored
- [ ] SWOT analysis included

## Gate Decision
- **PASS**: All mandatory + mode-specific requirements met
- **ITERATE**: Missing 1-2 items → ask @analyst to complete
- **ESCALATE**: Fundamental gaps → present options to user
