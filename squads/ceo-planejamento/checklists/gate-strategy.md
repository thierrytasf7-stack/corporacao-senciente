# Quality Gate: Strategy Phase

## Mandatory (all must pass)
- [ ] Clear objectives and goals defined
- [ ] User personas referenced
- [ ] Feature list with priorities (MoSCoW or RICE)
- [ ] Success metrics/KPIs defined
- [ ] Scope boundaries clear (in-scope vs out-of-scope)

## Standard Mode (4+ must pass)
- [ ] PRD or Epic document complete
- [ ] Non-functional requirements specified (performance, security, scalability)
- [ ] User stories outlined (high-level)
- [ ] Risk factors identified
- [ ] Business value articulated
- [ ] Dependencies mapped

## Comprehensive Mode (all must pass)
- [ ] Full PRD with all sections
- [ ] Detailed prioritization with scoring
- [ ] Phased delivery plan (MVP, v1, v2)
- [ ] Business case with ROI estimate
- [ ] Stakeholder alignment documented
- [ ] Go/no-go criteria defined
- [ ] CodeRabbit quality integration embedded

- **PASS**: All mandatory + mode-specific requirements met
