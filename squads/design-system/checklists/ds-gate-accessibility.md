# Quality Gate: Accessibility (WCAG AA)

## Color & Contrast
- [ ] Normal text contrast >= 4.5:1
- [ ] Large text (>=18px bold, >=24px regular) contrast >= 3:1
- [ ] UI components (borders, icons) contrast >= 3:1
- [ ] Focus indicators contrast >= 3:1
- [ ] Color is never the only information channel

## Keyboard
- [ ] All interactive elements reachable via Tab
- [ ] Logical focus order
- [ ] Visible focus indicator
- [ ] Escape closes modals/dialogs
- [ ] Enter/Space activates buttons
- [ ] Arrow keys for menus/tabs/radio groups

## Screen Reader
- [ ] Appropriate ARIA roles
- [ ] Labels on all inputs
- [ ] Icon-only buttons have aria-label
- [ ] Dynamic content uses aria-live
- [ ] Decorative elements have aria-hidden

## Motion
- [ ] prefers-reduced-motion supported
- [ ] No auto-play > 5s without controls
- [ ] No 3+ flashes/second

## Responsive
- [ ] Readable at 200% zoom
- [ ] Touch targets >= 44x44px
- [ ] No horizontal scroll at 320px
