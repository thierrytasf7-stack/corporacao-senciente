# Responsive Design Checklist

## Mobile (320px - 375px)
- [ ] Sem horizontal scroll
- [ ] Texto legivel sem zoom (>= 14px)
- [ ] Touch targets >= 44x44px
- [ ] Menu hamburger funcional
- [ ] Sidebar colapsada/hidden
- [ ] Tabelas scrollam horizontalmente ou adaptam
- [ ] Modais cabem na tela
- [ ] Formularios usam largura total
- [ ] Imagens redimensionam proporcionalmente

## Tablet (768px)
- [ ] Layout adapta entre mobile e desktop
- [ ] Sidebar pode ser toggle
- [ ] Grid ajusta colunas (2-3 ao inves de 4+)
- [ ] Conteudo principal nao fica espremido
- [ ] Touch-friendly (botoes e links com bom spacing)

## Desktop (1024px - 1920px)
- [ ] Layout usa espaco disponivel
- [ ] Sem linhas de texto > 80 caracteres
- [ ] Sidebar visivel por padrao
- [ ] Grid multi-coluna funcional
- [ ] Hover states presentes

## Cross-viewport
- [ ] Viewport meta tag presente e correta
- [ ] Sem media queries conflitantes
- [ ] Breakpoints consistentes no design system
- [ ] Orientacao landscape funciona
- [ ] Zoom ate 200% nao quebra layout
