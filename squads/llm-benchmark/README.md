# LLM Benchmark Squad

Squad de benchmark especializado em análise custo vs qualidade de modelos LLM, agentes e workflows.

## Agente

- **Metric** (Benchmark Analyst) - Analista de custo vs qualidade com foco Pareto 80/20

## Comandos

| Comando | Descrição |
|---------|-----------|
| `*benchmark-model {model}` | Custo e qualidade de um modelo |
| `*benchmark-agent {path}` | Analisar agente, identificar desperdício |
| `*benchmark-workflow {path}` | Analisar workflow completo |
| `*compare {A} vs {B}` | Comparar modelos custo vs qualidade |
| `*optimize-cost {target}` | Reduzir custo (Pareto) |
| `*optimize-quality {target}` | Melhorar qualidade |
| `*report` | Relatório completo |
