# Task: Create Component

> **Goal:** Criar um novo componente UI completo (Spec -> Code -> Story -> Test).
> **Role:** Orchestrator (Lead) -> Artist -> Auditor

## Instructions

1.  **Specification (Architect):**
    *   Definir nome, propósito e variantes do componente.
    *   Definir props e API.

2.  **Implementation (Artist):**
    *   Criar arquivo do componente (ex: `Button.tsx`).
    *   Usar APENAS tokens de design.
    *   Implementar estados (Hover, Focus, Active, Disabled).

3.  **Documentation (Artist):**
    *   Criar Storybook story (ex: `Button.stories.tsx`).
    *   Documentar uso no README do componente.

4.  **Audit (Blind Auditor):**
    *   Verificar contraste.
    *   Verificar navegação por teclado.
    *   Verificar labels ARIA.

5.  **Final Review:**
    *   Aprovar apenas se passar em todos os critérios do Visual Arete.
