# Structure Checklist

## Root Files
- [ ] `squad.yaml` presente na raiz
- [ ] `README.md` presente na raiz

## Required Directories
- [ ] `agents/` existe com pelo menos 1 .md
- [ ] `tasks/` existe com pelo menos 1 .md

## Optional Directories (se referenciados)
- [ ] `workflows/` existe se workflows em components
- [ ] `checklists/` existe se checklists em components
- [ ] `templates/` existe se templates em components
- [ ] `config/` existe se config files necessarios
- [ ] `data/` existe se data files necessarios
- [ ] `scripts/` existe se scripts necessarios
- [ ] `tools/` existe se tools necessarios

## Naming
- [ ] Todos files usam kebab-case
- [ ] Agents: .md extension
- [ ] Tasks: .md extension
- [ ] Workflows: .yaml extension
- [ ] Checklists: .md extension
- [ ] Templates: .md ou .yaml extension

## Clean State
- [ ] Sem diretorios vazios referenciados
- [ ] Sem arquivos temporarios (.tmp, .bak, ~)
- [ ] Sem files orfaos na raiz (exceto squad.yaml, README.md)
- [ ] Sem duplicatas
