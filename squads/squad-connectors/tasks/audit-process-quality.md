# Auditoria de Qualidade de Processo

## Objetivo
Avaliar a qualidade técnica e processual de um fluxo ou squad específico.

## Passos
1.  Selecionar o processo alvo (ex: Deploy, Code Review, Data Ingestion).
2.  Verificar adesão aos padrões definidos em `config/coding-standards.md` (se existir).
3.  Identificar pontos de falha manual.
4.  Verificar existência de testes automatizados para o processo.
5.  Gerar relatório de conformidade e sugestões de melhoria (hooks, validações).
