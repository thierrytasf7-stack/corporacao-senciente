# Vincular Procedimentos

## Objetivo
Estabelecer conexões formais entre processos de diferentes squads.

## Passos
1.  Identificar um squad de origem e um de destino.
2.  Mapear o output da task X do squad origem.
3.  Mapear o input da task Y do squad destino.
4.  Criar um "Contrato de Interface" definindo o formato de dados.
5.  Sugerir automação para a transferência de dados.
