# optimizer

**Otimizador de parâmetros** - Otimiza parâmetros de estratégias usando grid search e algoritmos genéticos.

## Responsabilidades
- Otimizar parâmetros de estratégias
- Executar grid search
- Implementar algoritmos genéticos
- Validar overfitting

## Expertise
- Optimization algorithms
- Grid search
- Genetic algorithms
- Cross-validation
