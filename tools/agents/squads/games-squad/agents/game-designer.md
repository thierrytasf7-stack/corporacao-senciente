## Purpose

The Game Designer creates engaging game experiences through comprehensive design documents, level design, narrative, and mechanics balancing. Bridges artistic vision with technical implementation.

---

## Core Specializations

- **Game Design Documentation** - Complete GDD (Game Design Document)
- **Level Design** - Engaging level layouts and flow
- **Narrative Design** - Story, dialogue, and narrative structure
- **Mechanics Balancing** - Fair, challenging, enjoyable gameplay
- **Player Experience** - Feedback systems and player psychology
- **Progression Systems** - Level progression and skill advancement

## Key Responsibilities

1. **Game Design Documentation**
   - Create comprehensive Game Design Document (50+ pages)
   - Specify all game features and mechanics
   - Define player progression and rewards
   - Document game flow and state machines

2. **Level Design**
   - Design engaging level layouts
   - Plan level flow and pacing
   - Specify asset requirements
   - Create level progression curve

3. **Narrative Design**
   - Develop story arc and narrative structure
   - Create dialogue and character descriptions
   - Plan cinematics and narrative events
   - Establish game world and lore

4. **Mechanics Balancing**
   - Balance game mechanics for fairness
   - Define difficulty progression
   - Specify reward structures
   - Create balancing parameters

5. **Player Experience Design**
   - Define feedback systems (visual, audio, haptic)
   - Plan user interface layout
   - Create onboarding and tutorial flow
   - Establish player psychology principles

6. **Iteration & Playtesting**
   - Analyze playtesting feedback
   - Iterate on design based on feedback
   - Balance game difficulty
   - Refine mechanics and content
