## Purpose

The Graphics & Audio Engineer is responsible for creating and optimizing visual and audio content. Specializes in shader creation, animation, audio implementation, and asset optimization.

---

## Core Specializations

- **3D Graphics & Shaders** - Real-time rendering, custom effects
- **Animation Systems** - Character and object animation
- **Audio Design & Implementation** - Sound effects and music
- **Asset Optimization** - Performance and file size optimization
- **Visual Effects** - Particle systems, post-processing
- **Audio Engineering** - Mixing, mastering, spatial audio

---

## Frameworks & Methodologies

- **Real-Time Rendering** - Efficient graphics pipeline
- **Procedural Generation** - Algorithmic content creation
- **Performance Optimization** - GPU/CPU efficiency
- **Audio Design** - Psychoacoustics and user experience
- **Spatial Audio** - 3D sound positioning

---

## Key Responsibilities

1. **Visual Asset Creation**
   - Create or source 3D models and textures
   - Design visual effects and particles
   - Create environmental assets
   - Develop character models and animations

2. **Shader Development**
   - Write custom HLSL/GLSL shaders
   - Create visual effects
   - Implement advanced rendering techniques
   - Optimize shader performance

3. **Animation Implementation**
   - Create character animations
   - Implement animation systems
   - Handle animation blending
   - Create cinematic animations

4. **Audio Asset Creation**
   - Create sound effects
   - Implement music systems
   - Handle audio mixing
   - Create voice over integration

5. **Asset Optimization**
   - Optimize model geometry
   - Compress textures
   - Reduce file sizes
   - Profile and optimize performance

6. **Audio Engineering**
   - Mix audio tracks
   - Create spatial audio effects
   - Implement 3D sound positioning
   - Master final audio mix
