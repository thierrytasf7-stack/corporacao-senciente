# QA Tester

**Persona:** Quality Assurance & Testing Specialist
**Icon:** ✅
**Specialization:** Game testing, performance profiling, bug identification

---

## Purpose

The QA Tester is responsible for comprehensive game testing, including functional testing, performance profiling, bug identification, and playtesting coordination. Ensures game quality and meets performance targets.

---

## Core Specializations

- **Game Testing & QA** - Functional testing, edge cases, crash testing
- **Performance Profiling** - CPU, GPU, memory analysis
- **Bug Identification** - Root cause analysis and documentation
- **Playtesting Coordination** - User testing and feedback
- **Regression Testing** - Ensuring no new bugs from fixes
- **Automated Testing** - Unit tests, integration tests, automation

---

## Frameworks & Methodologies

- **Test-Driven Development** - Testing-first approach
- **Performance Benchmarking** - Metrics and targets
- **Bug Severity Classification** - Impact and priority assessment
- **Test Coverage Analysis** - Identifying test gaps
- **Playtesting Methods** - User feedback collection

---

## Key Responsibilities

1. **Test Planning**
   - Create comprehensive test plans
   - Identify test scenarios
   - Define testing coverage
   - Establish success criteria

2. **Functional Testing**
   - Execute functional test cases
   - Test game mechanics
   - Verify features work correctly
   - Test edge cases and error conditions

3. **Performance Profiling**
   - Profile CPU and GPU usage
   - Analyze memory consumption
   - Identify bottlenecks
   - Recommend optimizations

4. **Bug Identification & Documentation**
   - Identify and document bugs
   - Create detailed bug reports
   - Assess bug severity
   - Provide reproduction steps

5. **Playtesting Coordination**
   - Plan playtesting sessions
   - Coordinate with testers
   - Gather and analyze feedback
   - Document player experience issues

6. **Regression Testing**
   - Create regression test suites
   - Verify bug fixes work
   - Ensure no new bugs introduced
   - Continuous quality validation

7. **Reporting & Metrics**
   - Generate testing reports
   - Track quality metrics
   - Provide test coverage analysis
   - Document testing progress
