## RPG (Role-Playing Games)

### Características Principais
- **Progressão de Personagem**: Sistema de níveis, habilidades e atributos
- **Narrativa Profunda**: História rica com escolhas que impactam o resultado
- **Customização**: Personalização de personagens, equipamentos e builds
- **Sistema de Combate**: Turnos ou tempo real com mecânicas táticas
- **Exploração**: Mundos abertos ou semi-abertos para descobrir

### Subgêneros
- **JRPG** (Japanese RPG): Final Fantasy, Dragon Quest
- **WRPG** (Western RPG): The Elder Scrolls, The Witcher
- **Action RPG**: Dark Souls, Diablo
- **Tactical RPG**: Fire Emblem, XCOM
- **MMORPG**: World of Warcraft, Final Fantasy XIV

### Template de Desenvolvimento
```
Core Systems:
- Character Stats (STR, DEX, INT, etc.)
- Experience/Leveling System
- Inventory Management
- Quest System
- Dialogue Trees
- Save/Load System

Gameplay Loop:
1. Explore → 2. Combat → 3. Loot → 4. Progress → 5. Repeat
```

## RTS (Real-Time Strategy)

### Características Principais
- **Gestão de Recursos**: Coleta e administração de materiais
- **Construção de Base**: Desenvolvimento de estruturas e defesas
- **Comando de Unidades**: Controle direto de exércitos
- **Estratégia em Tempo Real**: Decisões táticas instantâneas
- **Multiplayer Competitivo**: Foco em PvP estratégico

### Subgêneros
- **Classic RTS**: StarCraft, Age of Empires
- **Tower Defense**: Plants vs. Zombies, Defense Grid
- **MOBA**: League of Legends, Dota 2
- **Grand Strategy**: Civilization, Europa Universalis
- **Real-Time Tactics**: Company of Heroes

### Template de Desenvolvimento
```
Core Systems:
- Resource Management (Wood, Gold, Food, etc.)
- Unit Production
- Building Construction
- Tech Trees
- Fog of War
- AI Pathfinding

Gameplay Loop:
1. Gather Resources → 2. Build → 3. Research → 4. Expand → 5. Combat
```

## Simulation

### Características Principais
- **Realismo**: Simulação precisa de sistemas reais
- **Complexidade**: Múltiplas variáveis interconectadas
- **Gestão**: Administração de recursos, pessoas ou sistemas
- **Sandbox**: Liberdade criativa sem objetivos rígidos
- **Educacional**: Aprendizado através da experiência

### Subgêneros
- **Life Simulation**: The Sims, Animal Crossing
- **City Building**: SimCity, Cities: Skylines
- **Vehicle Simulation**: Flight Simulator, Euro Truck Simulator
- **Business Simulation**: RollerCoaster Tycoon, Planet Coaster
- **God Games**: Spore, Black & White

### Template de Desenvolvimento
```
Core Systems:
- Physics Engine
- Economic Models
- AI Behavior Systems
- Data Visualization
- Sandbox Tools
- Statistics Tracking

Gameplay Loop:
1. Plan → 2. Build/Manage → 3. Observe → 4. Adjust → 5. Expand
```

## Puzzle

### Características Principais
- **Resolução de Problemas**: Desafios lógicos e mentais
- **Progressão Gradual**: Dificuldade crescente
- **Mecânicas Simples**: Regras fáceis de entender
- **Satisfação**: Recompensa pela solução
- **Rejogabilidade**: Múltiplas soluções ou níveis

### Subgêneros
- **Match-3**: Candy Crush, Bejeweled
- **Physics Puzzle**: Portal, World of Goo
- **Logic Puzzle**: Sudoku, Picross
- **Platform Puzzle**: Baba Is You, The Witness
- **Hidden Object**: Mystery Case Files

### Template de Desenvolvimento
```
Core Systems:
- Level Editor
- Hint System
- Progress Tracking
- Achievement System
- Difficulty Scaling
- Undo/Redo Mechanics

Gameplay Loop:
1. Present Challenge → 2. Analyze → 3. Attempt Solution → 4. Learn → 5. Progress
```

## Action

### Características Principais
- **Reflexos Rápidos**: Reações em tempo real
- **Coordenação**: Habilidade motora e timing
- **Intensidade**: Ritmo acelerado e adrenalina
- **Combate Dinâmico**: Sistemas de luta fluidos
- **Desafio Físico**: Teste de habilidades do jogador

### Subgêneros
- **FPS** (First-Person Shooter): Call of Duty, Counter-Strike
- **Fighting Games**: Street Fighter, Tekken
- **Beat 'em Up**: Streets of Rage, Devil May Cry
- **Platformer**: Super Mario, Sonic
- **Bullet Hell**: Touhou, Ikaruga

### Template de Desenvolvimento
```
Core Systems:
- Input Handling (Low Latency)
- Physics Engine
- Collision Detection
- Animation System
- Audio Feedback
- Difficulty Balancing

Gameplay Loop:
1. React → 2. Execute → 3. Overcome → 4. Advance → 5. Master
```

## Indie

### Características Principais
- **Criatividade**: Conceitos únicos e experimentais
- **Recursos Limitados**: Desenvolvimento com orçamento reduzido
- **Visão Autoral**: Expressão pessoal dos desenvolvedores
- **Inovação**: Mecânicas não convencionais
- **Comunidade**: Relacionamento próximo com jogadores

### Subgêneros
- **Art Games**: Journey, Gris
- **Experimental**: Papers Please, Stanley Parable
- **Retro-Inspired**: Shovel Knight, Celeste
- **Narrative-Driven**: Firewatch, What Remains of Edith Finch
- **Minimalist**: Thomas Was Alone, LIMBO

### Template de Desenvolvimento
```
Core Systems:
- Modular Architecture
- Asset Pipeline Optimization
- Community Features
- Analytics Integration
- Localization Support
- Platform Adaptation

Gameplay Loop:
1. Experiment → 2. Iterate → 3. Polish → 4. Share → 5. Evolve
```

## Casual

### Características Principais
- **Acessibilidade**: Fácil de aprender e jogar
- **Sessões Curtas**: Gameplay em pequenos períodos
- **Interface Simples**: Controles intuitivos
- **Baixa Barreira de Entrada**: Não requer experiência prévia
- **Relaxante**: Experiência desestressante

### Subgêneros
- **Mobile Games**: Angry Birds, Clash of Clans
- **Social Games**: FarmVille, Words with Friends
- **Time Management**: Diner Dash, Cook-Out
- **Card Games**: Solitaire, Hearthstone
- **Idle Games**: Cookie Clicker, AdVenture Capitalist

### Template de Desenvolvimento
```
Core Systems:
- Touch/Mouse Controls
- Tutorial System
- Progress Persistence
- Social Features
- Monetization Integration
- Accessibility Options

Gameplay Loop:
1. Quick Start → 2. Simple Action → 3. Immediate Feedback → 4. Short Session → 5. Easy Return
```

## Hybrid Genres

### Características Principais
- **Fusão de Mecânicas**: Combinação de elementos de múltiplos gêneros
- **Inovação**: Criação de experiências únicas
- **Complexidade Balanceada**: Profundidade sem sobrecarga
- **Público Diverso**: Atração de diferentes tipos de jogadores
- **Evolução**: Adaptação e crescimento do medium

### Exemplos Populares
- **RPG + Action**: Mass Effect, The Witcher 3
- **Strategy + RPG**: Total War: Warhammer, Crusader Kings
- **Puzzle + Platform**: Portal, Baba Is You
- **Simulation + Strategy**: Cities: Skylines, Planet Coaster
- **Adventure + Survival**: Subnautica, Green Hell

### Template de Desenvolvimento
```
Core Systems:
- Modular Design
- Flexible Architecture
- Cross-Genre Balancing
- Player Choice Systems
- Adaptive Difficulty
- Multi-Layered Tutorials

Gameplay Loop:
1. Identify Core Loops → 2. Integrate Mechanics → 3. Balance Systems → 4. Test Cohesion → 5. Refine Experience
```

## Métricas e KPIs por Gênero

### RPG
- **Tempo de Sessão**: 45-120 minutos
- **Retenção D1**: 70%+
- **Retenção D7**: 40%+
- **Completion Rate**: 15-30%

### Casual
- **Tempo de Sessão**: 3-15 minutos
- **Retenção D1**: 80%+
- **Retenção D7**: 50%+
- **Sessions per Day**: 3-8

### Action
- **Input Lag**: <50ms
- **Frame Rate**: 60fps+
- **Learning Curve**: Steep but rewarding
- **Skill Ceiling**: High

### Simulation
- **Complexity**: Gradual introduction
- **Tutorial Length**: 15-30 minutes
- **Sandbox Freedom**: High
- **Realism vs Fun**: Balanced
