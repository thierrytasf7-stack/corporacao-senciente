# Analyze Performance Task

## Descrição
Realizar análise trimestral de performance envolvendo todos os agentes.

## Agentes Envolvidos
- **Analyst (Analista)**: Coleta e analisa dados
- **Strategist (Estrategista)**: Interpreta resultados e define próximos passos
- **Creative (Criativo)**: Avalia performance de ativos
- **Social Specialist (Especialista em Social)**: Analisa performance por canal

## Etapas
1. **Coleta de Dados (Analyst)**
   - Reunir métricas de todos os canais
   - Consolidar dados em dashboard
   - Identificar tendências e padrões

2. **Análise por Agente**
   - **Analyst**: Análise quantitativa e insights
   - **Strategist**: Alinhamento com objetivos estratégicos
   - **Creative**: Performance de ativos criativos
   - **Social Specialist**: Performance por canal e plataforma

3. **Reunião de Revisão**
   - Apresentação de resultados
   - Discussão de insights
   - Identificação de oportunidades

4. **Planejamento de Ações**
   - Strategist define prioridades
   - Creative planeja otimizações
   - Social Specialist ajusta estratégia de canais
   - Analyst configura novos KPIs

5. **Documentação e Compartilhamento**
   - Criar relatório final
   - Compartilhar com stakeholders
   - Definir próximos checkpoints
