# Create Content Calendar Task

## Descrição
Produzir um calendário de conteúdo semanal coordenado entre múltiplos agentes.

## Agentes Envolvidos
- **Strategist (Estrategista)**: Define temas e diretrizes
- **Creative (Criativo)**: Cria ativos de conteúdo
- **Analyst (Analista)**: Valida performance e sugere ajustes

## Etapas
1. **Definição de Tema (Strategist)**
   - Escolher tema semanal
   - Definir objetivos de conteúdo
   - Estabelecer diretrizes de tom e estilo

2. **Produção de Conteúdo (Creative)**
   - Criar conteúdo para cada canal
   - Adaptar formato para diferentes plataformas
   - Garantir consistência da mensagem

3. **Validação (Analyst)**
   - Analisar performance de conteúdo similar
   - Sugerir otimizações
   - Validar alinhamento com KPIs

4. **Aprovação Final (Strategist)**
   - Revisar calendário completo
   - Aprovar ou solicitar ajustes
   - Definir prioridades de publicação

5. **Distribuição (Social Specialist)**
   - Agendar publicações
   - Preparar materiais de apoio
   - Configurar monitoramento
