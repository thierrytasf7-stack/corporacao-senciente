# Launch Campaign Task

## Descrição
Coordenar o lançamento de uma campanha de marketing de 90 dias envolvendo múltiplos agentes.

## Agentes Envolvidos
- **Strategist (Estrategista)**: Define objetivos e KPIs
- **Creative (Criativo)**: Desenvolve ativos criativos
- **Analyst (Analista)**: Monitora métricas e performance
- **Social Specialist (Especialista em Social)**: Executa distribuição em canais sociais

## Etapas
1. **Planejamento (Strategist)**
   - Definir objetivos SMART
   - Estabelecer KPIs
   - Criar cronograma de campanha

2. **Criação (Creative)**
   - Desenvolver conceito criativo
   - Produzir ativos (imagens, vídeos, textos)
   - Validar com Strategist

3. **Análise (Analyst)**
   - Configurar rastreamento de métricas
   - Criar dashboards de performance
   - Estabelecer benchmarks

4. **Distribuição (Social Specialist)**
   - Planejar calendário de publicações
   - Executar lançamento em canais
   - Monitorar engajamento inicial

5. **Loop de Feedback**
   - Analyst reporta performance semanal
   - Strategist ajusta estratégia
   - Creative otimiza ativos
   - Social Specialist refina distribuição
