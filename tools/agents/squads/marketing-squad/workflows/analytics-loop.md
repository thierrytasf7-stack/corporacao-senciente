# Analytics Loop Workflow

## Descrição
Loop contínuo de análise de performance envolvendo todos os agentes.

## Agentes Envolvidos
- **Analyst (Analista)**: Coleta e analisa dados
- **Strategist (Estrategista)**: Interpreta resultados e define estratégia
- **Creative (Criativo)**: Otimiza ativos baseado em dados
- **Social Specialist (Especialista em Social)**: Ajusta estratégia de canais

## Fluxo de Trabalho

### Diário (Analyst)
- Coletar métricas de performance
- Identificar tendências emergentes
- Alertar sobre desvios significativos

### Semanal (Analyst + Strategist)
- **Análise de Performance**
  - Analyst apresenta dashboard semanal
  - Strategist avalia alinhamento com objetivos

- **Reunião de Ajustes**
  - Strategist define prioridades
  - Creative planeja otimizações
  - Social Specialist ajusta estratégia de canais

### Quinzenal (Creative)
- **Otimização de Ativos**
  - Analisar performance de ativos
  - Identificar oportunidades de melhoria
  - Criar novas versões baseadas em dados

### Mensal (Social Specialist)
- **Revisão de Canais**
  - Analisar performance por plataforma
  - Identificar canais de alto e baixo desempenho
  - Ajustar estratégia de distribuição

### Trimestral (Todos os Agentes)
- **Revisão Estratégica**
  - Analyst apresenta análise trimestral
  - Strategist avalia resultados gerais
  - Creative revisa portfólio de ativos
  - Social Specialist planeja estratégia para próximo trimestre
