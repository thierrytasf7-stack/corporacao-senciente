# Campaign Orchestration Workflow

## Descrição
Workflow de 90 dias para planejamento e execução de campanha envolvendo múltiplos agentes.

## Agentes Envolvidos
- **Strategist (Estrategista)**: Define estratégia e objetivos
- **Creative (Criativo)**: Desenvolve ativos criativos
- **Analyst (Analista)**: Monitora performance e fornece insights
- **Social Specialist (Especialista em Social)**: Executa distribuição

## Fases da Campanha

### Fase 1: Planejamento (Dias 1-15)
- **Strategist**
  - Definir objetivos SMART
  - Estabelecer KPIs
  - Criar cronograma detalhado

- **Creative**
  - Desenvolver conceito criativo
  - Criar mood board e guidelines
  - Validar com Strategist

- **Analyst**
  - Configurar rastreamento de métricas
  - Criar dashboards de campanha
  - Estabelecer benchmarks

### Fase 2: Produção (Dias 16-45)
- **Creative**
  - Produzir ativos para toda campanha
  - Criar variações para diferentes canais
  - Validar com Analyst

- **Analyst**
  - Analisar performance de conteúdo similar
  - Sugerir otimizações
  - Validar alinhamento com KPIs

- **Strategist**
  - Aprovar ativos finais
  - Definir prioridades de lançamento

### Fase 3: Execução (Dias 46-75)
- **Social Specialist**
  - Planejar calendário de lançamento
  - Executar distribuição em canais
  - Monitorar engajamento inicial

- **Analyst**
  - Monitorar métricas em tempo real
  - Identificar oportunidades de ajuste
  - Alertar sobre problemas

- **Strategist**
  - Tomar decisões baseadas em dados
  - Ajustar estratégia se necessário

### Fase 4: Otimização (Dias 76-90)
- **Analyst**
  - Análise profunda de performance
  - Identificar insights acionáveis

- **Creative**
  - Otimizar ativos baseado em dados
  - Criar variações para testar

- **Social Specialist**
  - Ajustar estratégia de distribuição
  - Focar em canais de alto desempenho

- **Strategist**
  - Definir recomendações para próximas campanhas
  - Documentar aprendizados

### Loop de Feedback Contínuo
- **Semanal**
  - Analyst reporta performance
  - Strategist ajusta estratégia
  - Creative otimiza ativos
  - Social Specialist refina distribuição

- **Quinzenal**
  - Reunião de revisão com todos os agentes
  - Ajustes de médio prazo
  - Planejamento de próximas semanas
